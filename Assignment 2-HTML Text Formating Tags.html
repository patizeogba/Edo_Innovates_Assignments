HTML Text Formatting Tags

s/n Tags Functions Notes
1 b Bold text This HTML element defines bold text, without any extra importance.
2 strong Important text This HTML element defines text with strong importance.
The content inside is typically displayed in bold.
3 i Italic text This HTML element defines a part of text in an alternate voice or mood.
The content inside is typically displayed in italic.
4 em Emphasis text This HTML element defines emphasized text. The content inside is typically displayed in italic.
5 mark Marked text This HTML element defines defines text that should be marked or highlighted.
6 small Smaller text This HTML element defines defines smaller text.
7 del Deleted text This HTML element defines text that has been deleted from a document.
Browsers will usually strike a line through deleted text.
8 ins Inserted text This HTML element defines a text that has been inserted into a document.
Browsers will usually underline inserted text.
9 sub Subscript text This HTML element defines subscript text.
10 sup Superscript text This HTML element defines superscript text.
